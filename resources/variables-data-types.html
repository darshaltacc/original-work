What Are Variables?

A variable is used to store data in a program. Each variable has a name and a value that can change while the program runs.

Example of Variables

let name = "Alex";
let age = 20;
let isStudent = true;

In this example,

  • age stores a number
  • name stores text
  • isStudent stores a value that is either true or false

Common Data Types

A data type tells the computer what kind of data the variable holds.

  • Number: integers or decimals, for example: 10, 3.14
  • String: text in quotes, for example: "Hello"
  • Boolean: true or false
  • Null: intentionally empty
  • Undefined: declared but not given a value
