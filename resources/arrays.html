What Is an Array or List?

An array (or list) stores multiple values in a single variable. Each value has an index that represents its position.

Creating a List

numbers = [1, 2, 3, 4, 5]
print(numbers[0])

Lists are zero-indexed, which means that the first element is at index 0.
