What Is a Function?

A function is a reusable block of code that performs a specific task. Functions help organize code and avoid repetition throughout your code.

Basic Function Example

def greet(name):
    print("Hello, " + name)

greet("Alex")

This function takes a parameter called name and prints a greeting using that value.

Parameters vs Arguments

  • Parameters are variables listed in the function definition
  • Arguments are the actual values passed to the function
