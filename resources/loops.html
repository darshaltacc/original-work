What Are Loops?

Loops allow code to run repeatedly. Instead of writing the same code many times, a loop can repeat it automatically.

For Loops

for i in range(5):
    print(i)

This loop runs 5 times and prints the numbers 0 through 4. i is the loop variable.

While Loops

count = 0

while count < 5:
    print(count)
    count += 1

A while loop runs as long as its condition remains true.

For vs While

  • For loops are used when you know how many times to repeat.
  • While loops are used when you repeat until a condition changes.
