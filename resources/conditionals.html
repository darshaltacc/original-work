What Are Conditionals?

Conditionals allow a program to make decisions. They let your code run different blocks of code depending on whether a condition is true or false.

Basic if/else Syntax

age = 16

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

The condition age >= 18 is checked first. If it is true, the if block runs. Otherwise, the else block runs.

Common Comparison Operators

  • == equal to
  • != not equal to
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to
