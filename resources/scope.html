What Is Scope?

Scopedetermines where a variable can be accessed in a program. It controls which parts of your code can see or modify a variable.

Local vs Global Scope

x = 10  # global variable

def example():
    x = 5   # local variable
    print(x)

example()
print(x)

The variable inside the function is local, while the one outside is global.

Here are mistakes you can avoid by keeping scope in mind

  • Assuming variables exist outside a function
  • Accidentally overwriting global variables
  • Using unclear variable names
