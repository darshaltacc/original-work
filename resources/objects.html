What Is an Object?

An object groups related data and behavior together in your code. Objects can be used to represent real-world things.

Object Example

person = {
    "name": "Alex",
    "age": 16,
    "student": True
}

print(person["name"])

This object stores related information about a person using key-value pairs.

What could you use objects for?

  • User accounts
  • Game characters
  • Products in a store
