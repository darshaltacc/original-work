What Is Error Handling?

Error handling allows a program to respond when something goes wrong instead of crashing.

Try/Except Example

try:
    number = int(input("Enter a number: "))
    print(10 / number)
except:
    print("Something went wrong")

This code prevents the program from crashing if invalid input is provided to it.

Common errors that you can prevent using try/catch

  • Division by zero
  • Invalid input types
  • Missing variables
